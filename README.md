# Week10CodingAssignment

In this assignment for PromineoTech, we were asked to use HTML, Bootstrap, and JavaScript to create a single page website.
  This website should contain a Bootstrap styled table representing data.
  It should also contain a Bootstrap styled form that allows a user to add a new row of data to the table when submitted.
  
This project was challenging and I did several errors in my JavaScript code along the way.

The end-result is a vehicle inventory that can be added to or deleted from.
